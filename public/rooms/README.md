# Room photos

Three room types; each type has multiple photos. The app uses these filenames (see `src/lib/data/rooms.ts`):

**Room type 1 — Jacuzzi Rooms** (3 photos)
- jacuzzi-1.png (main)
- jacuzzi-2.png, jacuzzi-3.png

**Room type 2 — Pool View Rooms** (3 photos)
- pool-view-1.png (main)
- pool-view-2.png, pool-view-3.png

**Room type 3 — Balcony with Garden View** (3 photos)
- balcony-garden-1.png (main)
- balcony-garden-2.png, balcony-garden-3.png

Use .png or .jpg; keep the extension consistent in `src/lib/data/rooms.ts`.
